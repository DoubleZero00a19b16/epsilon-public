## Overview

This folder contains all visual design assets produced for the **Epsilon** product, including logo concepts, banner designs, and the finalized brand identity files. Assets are organized into prototype and final directories to clearly separate exploration work from production-ready deliverables.

---

## Folder Structure

```
Design/
├── Logo_Prototypes/        # Iterative logo design versions
├── Banner_Prototypes/      # Iterative banner design versions
└── Logo_Final/             # Production-ready final logo files
```

---

## Logo Prototypes

The `Logo_Prototypes/` folder contains all iterative versions of the Epsilon logo produced during the design exploration phase. Each version represents a distinct design direction or refinement, covering variations in form, typography, color palette, and symbol usage.

These files are preserved as a reference for design decisions made throughout the process and are not intended for production use.

---

## Banner Prototypes

The `Banner_Prototypes/` folder contains multiple versions of banner designs developed for the Epsilon product. Similar to the logo prototypes, these represent different visual approaches and layout configurations explored before settling on a final direction.

These files serve as documentation of the design iteration process.

---

## Final Logo

The `Logo_Final/` folder contains the finalized logo files that serve as the official visual representation of the Epsilon product. These files are production-ready and should be used in all official materials, including the application UI, documentation, marketing, and any external-facing content.

Do not substitute prototype files in place of the assets found here.
